# tools

A personal collection of small tools and apps—each one lives in its own folder so you can open, copy, or deploy it independently.

## Layout

| Tool | Path | Notes |
|------|------|--------|
| **tasks** | [`tasks/`](tasks/) | CLI-style task list (static HTML/JS, no build). Open `tasks.html`; details in [`tasks/README.md`](tasks/README.md). |

Add new entries to the table when you drop in another project.

## Adding something new

1. Create a folder named after the tool (lowercase, short, no spaces).
2. Put the app files inside—static HTML/CSS/JS, a small CLI script, or whatever fits.
3. Update this README with a one-line description and link.

That keeps the repo easy to scan and keeps each tool self-contained.
