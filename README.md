# tools

A personal collection of small tools and apps. Each one lives in its own folder so it is possible to open, copy, or deploy each one independently.

## Layout

| Tool | Path | Notes |
|------|------|--------|
| **tasks** | [`tasks/`](tasks/) | CLI-style task list (static HTML/JS, no build). Open `tasks.html`; details in [`tasks/README.md`](tasks/README.md). |
| **timer** | [`timer/`](timer/) | Pomodoro / countdown timer. Open `timer.html`; details in [`timer/README.md`](timer/README.md). |
 
## Structure
 
```
tools/
  tools.html        ← launcher
  README.md
  tasks/
    tasks.html
    tasks.css
    tasks.js
    README.md
  timer/
    timer.html
    timer.css
    timer.js
    README.md
```
