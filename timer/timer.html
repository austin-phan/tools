Type a command, then Enter. Timer runs in the tab title so you can minimize it.

start [N] · start a work session (default 25 min)
break [N] · start a break (default 5 min)
pause · pause the current timer
resume · resume a paused timer
stop · stop and discard the current session
extend N · add N minutes to the current session
status · show current session info in feedback
log · open the session log
clear log · clear all session history
settings · change defaults
help · show this panel

↑ recall last command · ↓ clear · click empty page to focus input

A chime plays when the timer finishes (if your browser allows it).
